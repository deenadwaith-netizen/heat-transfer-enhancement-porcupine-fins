# Heat Transfer Enhancement in a Heat Exchanger Using Porcupine Fins and Al2O3-Water Nanofluid

## Overview
This project presents a CFD-based numerical investigation of heat transfer enhancement in a double-pipe heat exchanger using porcupine fins and Al2O3-water nanofluid.

The study compares three cases:
- conventional heat exchanger
- heat exchanger with porcupine fins using water
- heat exchanger with porcupine fins using Al2O3-water nanofluid

The goal was to evaluate whether porcupine-fin geometry and nanofluid application can improve thermal performance under turbulent flow conditions.

## Objective
The main objective was to investigate the thermal performance of a double-pipe heat exchanger equipped with porcupine fins and using Al2O3-water nanofluid as the working fluid.

## Methodology
- 3D geometry created in ANSYS 2019
- CFD simulation performed in ANSYS Fluent
- Pressure-based solver with realizable k-epsilon turbulence model
- Reynolds number range studied: 5000 to 7000
- Comparison based on:
  - Nusselt number
  - friction factor
  - thermal enhancement factor (TEF)
  - effectiveness

## Tools Used
- ANSYS DesignModeler
- ANSYS Fluent
- CFD
- Heat transfer analysis
- Numerical simulation
- Technical report writing

## Key Results
The study showed improved thermal performance when porcupine fins and Al2O3-water nanofluid were used.

Reported results include:
- maximum increase of **35.44% in Nusselt number**
- maximum thermal enhancement factor of **1.359**
- maximum effectiveness of **0.3177**

These results indicate that heat transfer enhancement was achieved, although with increased flow resistance.
